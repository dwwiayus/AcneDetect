# AcneDetect

AcneDetect adalah aplikasi web untuk mendeteksi tingkat keparahan jerawat dan memberikan rekomendasi skincare.

## Tech Stack

- Frontend: React + Vite
- Backend: Express.js
- Database: PostgreSQL

## Cara Menjalankan Project

### Backend
cd backend
npm install
npm run db:migrate
npm run dev

### Frontend
cd frontend
npm install
npm run dev
